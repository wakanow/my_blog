## Installation

```
$ git clone https://github.com/wakanow/my_blog.git
$ cd myblog
$ virtualenv venv
$ pip install -r requirements.txt
$ set FLASK_APP=manage.py
$ flask forge
$ flask run
* Running on http://127.0.0.1:5000/
```
